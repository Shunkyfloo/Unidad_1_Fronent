# Proyecto - Unidad 1 (Frontend)

## JavaScript
Este proyecto usa JavaScript en el navegador. El código está en `script.js` y se carga desde `index.html`.

## Node.js / npm (opcional)
Si querés usar paquetes (npm), primero instalá Node.js (LTS) desde `https://nodejs.org/`.

Luego, en la carpeta del proyecto:

```bash
npm init -y
```
