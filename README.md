# Proyecto - Unidad 1 (Frontend)

## JavaScript
Este proyecto usa JavaScript en el navegador. El código está en `script.js` y se carga desde `index.html`.

## API (backend) — un solo comando desde la raíz del proyecto

Instalá Node.js LTS desde `https://nodejs.org/`.

**Opción A — Git Bash o terminal (carpeta del proyecto):**

```bash
npm run install-api
npm run api
```

**Opción B — Git Bash:** doble clic no aplica; ejecutá `./iniciar-api.sh` (la primera vez instala dependencias sola).

**Opción C — Windows:** doble clic en `iniciar-api.bat` (instala si falta `node_modules` y levanta la API).

**Cursor / VS Code:** `Terminal` → `Run Task…` → **API: iniciar backend**.

La API queda en `http://localhost:3000`. Abrí el frontend con Live Server.
